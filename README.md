# 作业公布站 - homework-system

一个简单易用的作业公布平台，支持学生端查看作业和教师端管理作业，具备实时更新、公告栏展示、统计分析等功能。
作者：geyuqi

## 功能特点

### 学生端
- 实时查看作业
- 深色/浅色模式切换
- 查看相关链接（支持外链文件）
- 实时更新通知（WebSocket）
- 公告栏展示
- 美观的界面设计

### 教师端
-  发布新作业
-  编辑已有作业
-  删除作业
-  添加作业相关链接（支持外链文件）
-  编辑公告
-  访问统计查看
-  云盘快捷入口
-  实时同步到所有用户

### 统计页面
-  总访问次数统计
-  今日访问统计
-  不同设备统计
-  详细访问记录表格

## 技术栈

- 后端: Node.js + Express
- 实时通信: WebSocket
- 前端: 原生 HTML/CSS/JS
- 数据存储: JSON文件

## 宝塔面板部署

### 详细部署指南

请查看 [宝塔部署指南.md](./宝塔部署指南.md) 获取详细的部署步骤。

### 快速部署步骤

1. **上传文件**到宝塔网站目录
2. **安装依赖**：npm install
3. **PM2管理**：添加项目到PM2管理器
4. **配置网站**：添加站点并配置反向代理
5. **访问网站**：通过域名访问

### 环境要求

- Node.js 14.x 或更高（推荐 16.x / 18.x / 20.x）
- 宝塔面板最新版本
- PM2 管理器
